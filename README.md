Product Data Cleaning and Transformation using Excel Power Query

Project Overview

     This project focuses on cleaning, transforming, and formatting product data
using Excel Power Query and Excel formatting features. The objective was to improve 
data quality, ensure consistency, and prepare the dataset for analysis and reporting.
Data Cleaning Tasks Performed

Handling Missing Values

•	Filled or managed missing values in the Price column. 
•	Handled missing entries in the Category column to ensure data completeness.
 Correcting Inconsistent Data
•	Standardized inconsistent values in the Product Name column. 
•	Corrected inconsistent entries in the Category column. 
•	Identified and fixed typos and misspellings within category names.

Removing Duplicate Records

•	Removed duplicate rows from the dataset to maintain data accuracy and integrity. 

Splitting and Merging Columns

•	Split the Product ID column to extract: 
o	Manufacturing Date 
o	Country Code 
•	Merged Brand Name and Product Name columns to create a new column called Product Brand. 

Removing Unnecessary Characters

•	Cleaned text fields by removing unwanted or unnecessary characters. 

Data Formatting
Number Formatting
•	Formatted the Price column as Currency. 
•	Formatted the Manufacturing Date column in DD-MM-YYYY (Day-Month-Year) format. 

Conditional Formatting

Price Column
•	Applied Data Bars with a blue color scale to visually represent product prices. 

Category Column
•	Applied conditional formatting to highlight cells containing "Electronics" in red color for easy identification. 
Tools Used
•	Microsoft Excel 
•	Excel Power Query 
•	Conditional Formatting 
•	Data Transformation and Cleaning Techniques 

Outcome
The dataset was successfully cleaned, standardized, and formatted, resulting in:
•	Improved data quality and consistency. 
•	Removal of duplicates and errors. 
•	Enhanced readability through proper formatting. 
•	Better visualization of key information using conditional formatting. 
•	A structured dataset ready for further analysis and reporting.
